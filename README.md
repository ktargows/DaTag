# DaTag
Applying Hierarchical edge bundling to visualize the relationships among Datasets, Attributes, and user-defined Tags.

# Connections defined in the "readme-flare-imports.jon"
  1. Databases --> Tags
  2. Tags --> Attributes
  3. Attrbutes --> Databases
